# Portfolio

A personal portfolio website built with React 19, TypeScript, Framer Motion, and Tailwind CSS v4.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** — build tooling
- **Tailwind CSS v4** — styling
- **Framer Motion** — animations
- **React Router DOM v7** — hash-based routing
- **Formspree** — contact form handling

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Deploys to GitHub Pages via the `gh-pages` branch:

```bash
npm run deploy
```
